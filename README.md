# Hospital Ontology

An OWL ontology modelling a hospital management system including 
Patients, Doctors, Diagnoses, and Treatments.

## How to Use
- Download the .owl file and open in Protégé
- Run HermiT reasoner to check consistency
- Use the SPARQL tab to query patient data

## Reuse
This ontology can be reused by any healthcare system 
needing to model patient-treatment relationships.
